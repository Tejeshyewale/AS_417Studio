## 🌐 Deploy for Free — Step by Step

### Option A: Vercel (Recommended — Fastest)

1. Go to [vercel.com](https://vercel.com) and sign up with GitHub
2. Click **"Add New Project"**
3. Upload your `as417studio.html` file
4. Click **Deploy**
5. Your site is live in 2 minutes ✅

**Free custom domain on Vercel:**  
`as417studio.vercel.app` — free forever

---

### Option B: Netlify (Also Free)

1. Go to [netlify.com](https://netlify.com)
2. Drag and drop your HTML file onto the deploy area
3. Done — instant live link ✅

---

### Option C: GitHub Pages (Free)

1. Create a new GitHub repository named `as417studio`
2. Upload `as417studio.html` and rename it to `index.html`
3. Go to **Settings → Pages → Source → main branch**
4. Your site is live at `yourusername.github.io/as417studio` ✅

## 🌍 Custom Domain Setup (Optional)

Want `as417studio.com` or `as417studio.in`?

### Step 1 — Buy Domain
- **Namecheap** → `namecheap.com` (cheapest, ~₹700/year for `.com`)
- **GoDaddy India** → `godaddy.com/in` (~₹800/year)
- **BigRock** → `bigrock.in` (~₹600/year for `.in`)

### Step 2 — Connect to Vercel
1. In Vercel dashboard → go to your project → **Settings → Domains**
2. Add your domain (e.g. `as417studio.com`)
3. Vercel will give you DNS records (two values: `A` and `CNAME`)
4. Go to your domain provider → **DNS Settings**
5. Add those records
6. Wait 10–30 minutes → your custom domain is live ✅

## 📬 How Contact Form Works

When someone fills the form and clicks **Send Message**:
- It opens their email app with pre-filled subject + message
- Email goes directly to your inbox
- No backend or server needed

**To use a proper form backend (free):**
- Sign up at [formspree.io](https://formspree.io)
- Get your form endpoint URL
- Replace the `submitForm()` function action with the Formspree URL
